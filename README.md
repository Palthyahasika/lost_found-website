# lost_found-website
Lost &amp; Found Portal is a full-stack web application built using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB. It enables users to report lost and found items, browse listings, search items, and manage records through a secure authentication system and an admin dashboard.
 🧳 Lost & Found Portal

## 📖 Overview

**Lost & Found Portal** is a full-stack web application designed to help users report, search, and manage lost and found items through a centralized digital platform.

The system provides a simple and efficient way for users to post lost or found belongings, browse listings, and search for items, reducing the reliance on informal communication channels and improving the chances of item recovery.

```text

      ↓

      ↓
```

## 🚀 Installation & Setup

### Clone Repository

```bash
git clone https://github.com/your-username/lost-found-portal.git
cd lost-found-portal
```

### Install Dependencies

```bash
npm install
```

### Environment Variables

Create a `.env` file:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```

### Run Application

```bash
npm start
```

or

```bash
npm run dev
```
